# Teaplc-rs

A Rust-based re-implementation of the [Teapl compiler](https://github.com/hxuhack/compiler_project.git).

## Quick Start

```
cargo run -- --input /Users/yisun/Projects/compiler_pj/tests/progs/dfs.tea
```
